keri.host chat

Conversational AI assistant for the KERI ecosystem

What is this?

A conversational AI assistant that answers questions about KERI, CESR, and ACDC grounded in the official specifications. Features hybrid search, multi-turn conversation, inline citations, and file analysis.

Use Cases

  • Spec compliance review for KERI implementations
  • Architecture guidance for decentralized identity systems
  • Protocol Q&A on CESR encoding and ACDC credentials
  • Key management and rotation pattern reference

Deploy Your Own

Launch a private instance in your own AWS account. The stack provisions a Bedrock knowledge base, ingestion pipeline, chat API, and this frontend.

Setup Requirements

After deploying the stack, first-time Anthropic model users must submit use case details:

  1. Open the Amazon Bedrock console in your deployment region
  2. Go to Model catalog in the left sidebar
  3. Click “Submit use case details” in the yellow banner at the top
  4. Fill in the short form (company name and brief use case description)

Access is typically granted within a few minutes. Without this step, the chat API will return a 503 error.

Get Access

This instance is IP-restricted. Call 630-788-5304 to have your access activated.

MCP Server

Every deployment includes a hosted MCP endpoint for use with Claude Code and other MCP clients. Add this to your Claude Code config:

{
  "mcpServers": {
    "keri-chat": {
      "type": "url",
      "url": "https://<your-domain>/mcp"
    }
  }
}
